## 🌍 PT-BR

Resource criada para conectar scripts da [Bjorn Store](https://discord.gg/PJRRHQtavX) às frameworks e dependências,
permitindo a integração com qualquer resource ou framework externa.


## 📌 Instalação:

1. [Baixe o ultima versão do script](https://github.com/Andrade001/BjornBridge/releases).
2. Coloque-o na pasta `resources` do seu servidor.
3. Abra seu `server.cfg` e adicione `ensure BjornBridge` antes de qualquer outro script da loja.


## ⚙️ Frameworks suportadas

- vRP
- vRPex
- Creative (V5, V6, Network e Enchanted)
- Esx (Em testes)
- QBcore (Em testes)
- Qbox (Em testes)
- Tnet (Em testes)
